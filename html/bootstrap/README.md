## Basic HTML site with bootstrap
Bootstrap enhanced version of HTML template. 

1. Look at html-template as a starting point. All pre-requisites, environment and process is the same
1. Add link to bootstrap css and scripts in header section using latest version/link in https://getbootstrap.com
   ```html 
    <link
      href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
      rel="stylesheet"
      integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH"
      crossorigin="anonymous"
    />
    <script
      src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
      integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz"
      crossorigin="anonymous"
    ></script>
   ```
1. Use bootstrap components, grid, formatting etc.
